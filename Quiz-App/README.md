# Quiz App 🎯

A simple command-line Quiz Application built using Python.

This project presents multiple-choice questions (MCQs) to the user, evaluates answers, calculates the score, and displays the final result based on performance.

## Features

* 📚 Multiple Choice Questions (MCQs)
* ✅ Instant Answer Validation
* 🎯 Score Tracking
* 📊 Percentage Calculation
* 🏆 Pass/Fail Evaluation
* 🔢 Question Counter
* 💡 Displays Correct Answer for Incorrect Responses
* 🔤 Case-Insensitive User Input

## Technologies Used

* Python 3
* Lists
* Dictionaries
* Loops
* Conditional Statements
* User Input Handling

## How It Works

1. The quiz displays one question at a time.
2. The user selects an answer (A/B/C/D).
3. The program checks whether the answer is correct.
4. The current score and percentage are displayed.
5. The quiz continues until the user chooses to stop.
6. A final performance summary is shown.
