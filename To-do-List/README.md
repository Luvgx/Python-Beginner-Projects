# 📝 To-Do List (CLI Application)

A simple Command Line Interface (CLI) To-Do List application built with Python. This project allows users to manage their daily tasks by adding, viewing, updating, and removing tasks directly from the terminal.

## 🚀 Features

* ➕ Add one or multiple tasks
* 📋 View all tasks with numbering
* ✏️ Update existing tasks
* ❌ Remove completed or unwanted tasks
* ✅ Input validation for invalid task numbers
* 📌 Handles empty task lists gracefully
* 🔄 Interactive menu-driven interface
* 👋 Clean exit message

## 🖥️ How It Works

When the program starts, a menu is displayed:

```text
1. Add Task
2. View Tasks
3. Update Task
4. Remove Task
5. Exit
```

### Add Task

Allows users to add one or more tasks.

### View Tasks

Displays all current tasks with numbering.

### Update Task

Updates a task by selecting its task number.

### Remove Task

Removes a task by selecting its task number.

### Exit

Closes the application safely.
